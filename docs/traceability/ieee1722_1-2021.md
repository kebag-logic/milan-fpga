# Traceability — IEEE 1722.1-2021 (ATDECC: ADP / ACMP / AECP+AEM)

Part of [`docs/SPEC_TRACEABILITY.md`](../SPEC_TRACEABILITY.md). Legend and
conventions live there. Clause numbers are from IEEE Std 1722.1-2021 (verified
against the PDF in `/home/alex/standards/1722.1-2021.pdf`). Milan v1.2
overrides/deltas are NOT repeated here — see
[`milan-v12.md`](milan-v12.md); a row marked *(Milan overrides)* means the
1722.1 base behavior is intentionally replaced and the Milan file carries the
authoritative row.

Verification tokens: `RTL <dir>` = `tb/verilator/<dir>`; `CERT` = bench behave
recreation feature; `SILICON` = wire-proven on ARTY/AX7101 (see
`docs/MILAN_COMPLIANCE_GAPS.md` / `docs/findings/HANDOVER.md`); `tsn_gen:` =
YAML model under `/home/alex/tsn-gen/protocols/` consumable by packet_gen
(generate + fuzz + decode on the wire). `NO MODEL` = tsn_gen model to author.
